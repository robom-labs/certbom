# 변경 기록

## 0.5.0 - 2026-07-16

- 공식 시험 카탈로그를 10개에서 97개로 늘리고, 8개 출처의 2026 일정 70개 시험을 교차 확인했습니다.
- 현재 접수 중·접수 예정·시험 예정 순 정렬, 약칭 검색, 상태·분야 필터와 12개씩 더 보기를 추가했습니다.
- 접수 기간의 종료일을 카드·상세·ICS·Google Calendar에 보존하고 공통 회차 일정은 홈에서 묶어 표시합니다.
- 운영 키가 없는 상태의 `공식 API` 표기와 반복 미연결 문구를 없애고 출처·마지막 검증일을 명시했습니다.
- 시험별 준비물 체크 식별자를 분리해 다른 시험의 체크 상태가 함께 바뀌던 문제를 수정했습니다.

## 0.4.1 - 2026-07-16

- Vercel Git 자동 배포에서도 실제 main 커밋 SHA가 설정 화면에 표시되도록 보완했습니다.
- Vercel 프로젝트 연결 파일과 OIDC 환경 파일이 저장소에 포함되지 않도록 고정했습니다.

## 0.4.0 - 2026-07-16

- 자격증 검색·상세·agenda·공식 준비물·추천 3+7을 제공하는 첫 PWA를 추가했습니다.
- 관심 시험, 준비물 체크, 링크·ICS·캘린더 공유를 로컬 우선 방식으로 추가했습니다.
- 공식 출처 레지스트리, Q-Net 응답 어댑터, Supabase RLS 마이그레이션과 CI를 추가했습니다.
