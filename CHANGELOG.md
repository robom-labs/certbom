# 변경 기록

## 0.6.3 - 2026-07-16

- 홈 "곧 시험" 필터가 시험 당일의 날짜 전용 시험과 진행 중인 기간형 시험을 제외하던 문제를 `eventRelevantUntil` 기준(KST 23:59:59)으로 판정해 수정했습니다.
- 시험 찾기 화면의 검색어·필터를 `#find?q=…&filter=…` URL에 replaceState로 동기화해 상세에서 뒤로 가기 후에도 상태가 유지됩니다.
- Q-Net source 동기화에서 4xx·비JSON 같은 결정적 실패는 재시도 없이 즉시 중단하고, 네트워크 오류·5xx·429만 재시도합니다.

## 0.6.2 - 2026-07-16

- Q-Net 국가자격 시험일정 API의 현재·다음 연도 pagination, resultCode, totalCount, 날짜 형식과 안정 ID를 검증하는 운영 adapter를 추가했습니다.
- 0건·대량 삭제·기간 역전·31일 초과 과거 이동·첫 스냅샷을 자동 공개하지 않고 단일 검토 이슈로 보내는 anomaly gate를 추가했습니다.
- KST 연도 전환과 400일 시간 이동, scheduler heartbeat 경계를 자동 검사해 2027년 이후에도 대상 연도가 코드 수정 없이 이동하도록 고정했습니다.
- 일일 source workflow는 Q-Net 운영 키가 없을 때 가짜 API 연결을 표시하지 않고 `BLOCKED_EXTERNAL`로 기록하며, 정상 복구 시 기존 운영 이슈를 자동으로 닫습니다.
- 데이터가 달라질 때만 last-known-good를 갱신하고, API 0건·오류 때 기존 정상 스냅샷을 덮어쓰지 않도록 했습니다.

## 0.6.1 - 2026-07-16

- 홈의 전체 시험·현재 접수·곧 시험 요약을 48px 이상 실제 필터 버튼으로 바꾸고, URL 상태와 선택 결과 목록을 바로 아래에 표시합니다.
- 여러 시험이 묶인 공통 회차는 첫 시험으로 임의 이동하지 않고 포함 시험 전체를 펼친 뒤 선택하도록 수정했습니다.
- 상세에서 홈·시험 찾기·내 일정 출처와 홈 필터·스크롤을 보존해 뒤로 가기 흐름을 복구했습니다.
- 준비물 전체 개수·완료·필수 미완료 진행률, 카테고리, 필수·권장·공식 확인 필요, 기준 버전·확인일을 표시합니다.
- 기존 `certbom-preparation-v1` 체크를 시험·준비물 버전별 안정 ID인 `certbom-preparation-v2`로 보존 이전하고 저장 실패 안내를 추가했습니다.
- Web Share 취소·clipboard 실패와 ICS object URL 수명을 안전하게 처리하고 Chromium·WebKit P0 회귀 검사를 추가했습니다.
- 네이티브 알림은 기존 자격증봄 예약을 취소한 뒤 한 건만 재예약하고, 날짜 전용 일정은 전날 KST 오전 안내로 처리하며 일정이 없을 때 임의 1분 알림을 만들지 않습니다.

## 0.6.0 - 2026-07-16

- immutable 로봄 패밀리 정본 `5a63eab4f6930837f9877efd187562b85530a81a`에서 생성한 토큰·봄 wordmark·선형 SVG 아이콘·기능 플래그·인증 설정과 hash lock을 적용했습니다.
- 48px 이상·safe-area 대응 하단 메뉴와 5개 패밀리 앱, 설치·업데이트, 지원·개인정보, 전체 앱 메타를 제공하는 설정 화면을 추가했습니다.
- `beforeinstallprompt` 기반 PWA 설치 CTA와 iOS Safari 홈 화면 추가 안내, 0.6.0 서비스워커 캐시를 연결했습니다.
- OAuth 준비 알림을 게스트 우선 공급자 중립 상태 adapter로 교체하고, 동의 기본 꺼짐 noop 분석 adapter를 연결했습니다.
- localStorage 접근·쓰기 실패를 격리하고 저장 흐름, 설치, 인증, 분석 회귀 테스트를 추가했습니다.
- Expo SDK 57 독립 네이티브 앱에 97개 오프라인 카탈로그, 관심 시험 저장, 로컬 알림, 공식 링크와 `certbom` 딥링크를 추가했습니다.

## 0.5.0 - 2026-07-16

- 공식 시험 카탈로그를 10개에서 97개로 늘리고, 8개 출처의 2026 일정 70개 시험을 교차 확인했습니다.
- 현재 접수 중·접수 예정·시험 예정 순 정렬, 약칭 검색, 상태·분야 필터와 12개씩 더 보기를 추가했습니다.
- 접수 기간의 종료일을 카드·상세·ICS·Google Calendar에 보존하고 공통 회차 일정은 홈에서 묶어 표시합니다.
- 운영 키가 없는 상태의 `공식 API` 표기와 반복 미연결 문구를 없애고 출처·마지막 검증일을 명시했습니다.
- 시험별 준비물 체크 식별자를 분리해 다른 시험의 체크 상태가 함께 바뀌던 문제를 수정했습니다.
