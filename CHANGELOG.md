# 변경 기록

## Unreleased

## 0.8.1 - 2026-07-23

- Expo SDK 57의 네이티브 의존성을 공식 호환 패치 버전으로 올리고 Android versionCode와 iOS buildNumber를 7로 올렸다.

## 0.8.0 - 2026-07-23

- 0.8.0: 네이티브 앱에서 관심 시험을 여러 개 기기에 저장하고, 시험별 로컬 알림은 해당 시험의 이전 알림만 교체하도록 수정했다.
- 0.8.0: 사용자가 요청하지 않은 5분 뒤 임의 알림을 제거하고, 알림은 정확히 예약할 수 있는 미래 일정에만 소리와 함께 예약한다.
- 0.8.0: 실제 앱 서명이 없는 빈 도메인 인증 파일과 충돌하던 Android App Link 자동 검증·iOS Universal Link 선언을 제거하고, 검증 가능한 `certbom://` 딥링크만 유지했다. Android versionCode와 iOS buildNumber를 6으로 올렸다.

- 네이티브 Production 빌드를 Android API 36·AAB로 고정하고 지원·개인정보 주소와 Expo 조직 소유자를 명시했다.
- 앱 설명의 고정된 시험 개수 문구를 제거하고 설정 검사에서 웹·네이티브 버전, API 수준, AAB와 법적 링크를 함께 검증하도록 했다.

## 0.7.2 - 2026-07-17

- 한국생산성본부(KPC, ITQ·GTQ 등 10개)와 한국공인회계사회 AT자격시험(FAT·TAT 등 4개)에도 공식 규정을 웹에서 확인한 준비물을 추가했다. KPC는 원본 신분증(대학생 학생증 불인정)·사진 부착 수험표, AT는 신분증·수험표(대면)·필기구·일반계산기·부정행위 2년 정지 안내를 출처와 함께 제공한다. 공식 준비물 보유 시험이 82개에서 96개(97개 중)로 늘었다.

## 0.7.1 - 2026-07-17

- Q-Net 국가기술자격·국가전문자격, 데이터자격검정, 대한상공회의소 시험 82개에 공식 규정을 웹에서 직접 확인한 준비물(신분증·수험표·필기구·계산기 허용군·반입 금지 물품)과 공식 출처 링크를 추가했습니다. 이제 이 시험들은 "확인 중" 안내 대신 필수·금지 표시가 붙은 구체적 준비물을 보여줍니다.
- 공식 확인 준비물에 입실 마감·최신 요강 최종 확인 일반 안내를 함께 두어, 검증된 항목과 일반 안내를 구분하면서도 빈손으로 돌려보내지 않습니다.
- 이전 일반 체크리스트(general-v1·general-v2)에서 체크한 항목이 새 출처 검증 항목으로 이어지도록 안정 ID 이전을 보강했습니다.

## 0.7.0 - 2026-07-17

- 공식 세부 준비물이 구조화되지 않은 시험도 신분증·수험표·필기구·실기 도구·금지 물품·입실 확인을 포함한 일반 준비 체크리스트를 제공하도록 개선했습니다.
- 공식·교차 확인 참고·일반 준비의 근거 수준을 데이터와 화면에서 구분해 일반 안내가 공식 확정 정보처럼 보이지 않게 했습니다.
- 하단 메뉴에 독립 시험 달력을 추가하고 접수·시험·수험표·발표·변경·취소 일정을 월간 달력과 날짜별 agenda로 확인할 수 있게 했습니다.
- 휴대폰에서는 달력 아래 일정으로 바로 이동하고, 태블릿에서는 달력과 날짜별 일정을 나란히 사용하는 적응형 화면을 추가했습니다.
- 달력의 월·날짜·관심 시험·이벤트 필터를 URL에 보존해 상세 화면을 다녀오거나 새로고침해도 같은 위치로 돌아오게 했습니다.
- 웹·core·source adapter·Android·iPhone 버전을 0.7.0으로 맞추고 네이티브 build 번호를 올렸습니다.
- 네이티브 목록을 태블릿 너비에서 중앙 정렬하고 iPad·Android 태블릿의 가로 회전을 허용했습니다.
- 네이티브 설정 검사에 회전·iPad 지원 계약을 추가하고 build 번호를 5로 올렸습니다.

## 0.6.3 - 2026-07-16

- 홈 "곧 시험" 필터가 시험 당일의 날짜 전용 시험과 진행 중인 기간형 시험을 제외하던 문제를 `eventRelevantUntil` 기준(KST 23:59:59)으로 판정해 수정했습니다.
- 시험 찾기 화면의 검색어·필터를 `#find?q=…&filter=…` URL에 replaceState로 동기화해 상세에서 뒤로 가기 후에도 상태가 유지됩니다.
- Q-Net source 동기화에서 4xx·비JSON 같은 결정적 실패는 재시도 없이 즉시 중단하고, 네트워크 오류·5xx·429만 재시도합니다.

## 0.6.2 - 2026-07-16

- Q-Net 국가자격 시험일정 API의 현재·다음 연도 pagination, resultCode, totalCount, 날짜 형식과 안정 ID를 검증하는 운영 adapter를 추가했습니다.
- 0건·대량 삭제·기간 역전·31일 초과 과거 이동·첫 스냅샷을 자동 공개하지 않고 단일 검토 이슈로 보내는 anomaly gate를 추가했습니다.
- KST 연도 전환과 400일 시간 이동, scheduler heartbeat 경계를 자동 검사해 2027년 이후에도 대상 연도가 코드 수정 없이 이동하도록 고정했습니다.
- 일일 source workflow는 Q-Net 운영 키가 없을 때 가짜 API 연결을 표시하지 않고 `BLOCKED_EXTERNAL`로 기록하며, 정상 복구 시 기존 운영 이슈를 자동으로 닫습니다.
- 데이터가 달라질 때만 last-known-good를 갱신하고, API 0건·오류 때 기존 정상 스냅샷을 덮어쓰지 않도록 했습니다.

## 0.6.1 - 2026-07-16

- 홈의 전체 시험·현재 접수·곧 시험 요약을 48px 이상 실제 필터 버튼으로 바꾸고, URL 상태와 선택 결과 목록을 바로 아래에 표시합니다.
- 여러 시험이 묶인 공통 회차는 첫 시험으로 임의 이동하지 않고 포함 시험 전체를 펼친 뒤 선택하도록 수정했습니다.
- 상세에서 홈·시험 찾기·내 일정 출처와 홈 필터·스크롤을 보존해 뒤로 가기 흐름을 복구했습니다.
- 준비물 전체 개수·완료·필수 미완료 진행률, 카테고리, 필수·권장·공식 확인 필요, 기준 버전·확인일을 표시합니다.
- 기존 `certbom-preparation-v1` 체크를 시험·준비물 버전별 안정 ID인 `certbom-preparation-v2`로 보존 이전하고 저장 실패 안내를 추가했습니다.
- Web Share 취소·clipboard 실패와 ICS object URL 수명을 안전하게 처리하고 Chromium·WebKit P0 회귀 검사를 추가했습니다.
- 네이티브 알림은 기존 자격증봄 예약을 취소한 뒤 한 건만 재예약하고, 날짜 전용 일정은 전날 KST 오전 안내로 처리하며 일정이 없을 때 임의 1분 알림을 만들지 않습니다.
