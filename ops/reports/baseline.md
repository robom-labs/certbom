# 자격증봄 기준선

## 제품 판단

- 자격증봄은 원서접수 시스템을 대체하지 않고, 공식 일정을 쉽게 찾고 놓치지 않게 하는 편의 서비스다.
- 로그인 없이 검색, 추천, 상세, 공유, 공식 접수처 이동을 제공한다.
- 운영 날짜는 공식 API나 공식 공고로 확인된 경우에만 노출한다. 샘플과 추정 날짜는 노출하지 않는다.
- 상시 시험에는 임의 D-day를 만들지 않고 공식 접수처로 안내한다.

## 0.5.0 공식 데이터 범위

- 카탈로그는 공식 출처 8곳의 시험 97개이며, 그중 70개 시험에 2026년 확정 일정이 있다.
- Q-Net 국가기술자격 3회, 국가전문자격 공식 캘린더 PDF, 한국사 제79~81회, K-DATA, KPC, AT, 대한상공회의소 일정을 2026년 7월 16일에 교차 확인했다.
- Q-Net 공공데이터 API 계약은 유지하지만 운영 서비스키가 없어 현재 앱 데이터에는 사용하지 않는다. 화면에도 `공식 API`로 표시하지 않는다.
- 상시검정과 공고형 시험은 날짜를 만들지 않고 공식 접수처 또는 최신 공고로 연결한다.

## 전후 측정

- 시험 수. 10개에서 97개로 증가했다.
- 확정 일정 보유 시험. 1개에서 70개로 증가했다.
- 공식 출처. 4개 레지스트리에서 실제 노출 출처 8개로 증가했다.
- 모바일 320×568. 검색 CTA가 첫 화면 안에 있고 검색 입력 높이는 58px, 주요 버튼은 최소 48px다.
- 번들. 자바스크립트 gzip 70.65KB에서 77.92KB로 7.27KB 증가했다.
- 브라우저. 320×568, 390×844, 1280×800에서 가로 넘침과 콘솔 오류가 없었다.

## 위험과 대응

- 외부 소스 변경은 런타임 스키마와 fingerprint로 감지하고, 0건·대량 삭제·필드 변경은 자동 공개하지 않는다.
- PWA 캐시는 앱 셸만 저장하고 공식 접수처 요청은 네트워크 전용으로 둔다.
- OAuth, Web Push, 원격 동기화는 콘솔 키 없이는 활성화하지 않는다. 로컬 관심 저장과 Web Share·ICS는 계속 동작한다.
