# 자격증봄 기준선

## 제품 판단

- 자격증봄은 원서접수 시스템을 대체하지 않고, 공식 일정을 쉽게 찾고 놓치지 않게 하는 편의 서비스다.
- 로그인 없이 검색, 추천, 상세, 공유, 공식 접수처 이동을 제공한다.
- 운영 날짜는 공식 API나 공식 공고로 확인된 경우에만 노출한다. 샘플과 추정 날짜는 노출하지 않는다.
- 상시 시험에는 임의 D-day를 만들지 않고 공식 접수처로 안내한다.

## 초기 공식 데이터 범위

- 한국사능력검정시험 제79회의 2026년 접수 잔여석, 수험표, 시험, 결과 일정을 공식 홈페이지에서 확인했다.
- Q-Net 국가자격 시험일정 조회 API의 JSON 계약과 안정 ID 구성 필드를 고정했다. 운영 호출은 서비스키 등록 후 시작한다.
- 대한상공회의소와 국가공무원 일정은 공식 링크만 제공하며 자동 수집 범위에 포함하지 않는다.

## 위험과 대응

- 외부 소스 변경은 런타임 스키마와 fingerprint로 감지하고, 0건·대량 삭제·필드 변경은 자동 공개하지 않는다.
- PWA 캐시는 앱 셸만 저장하고 공식 접수처 요청은 네트워크 전용으로 둔다.
- OAuth, Web Push, 원격 동기화는 콘솔 키 없이는 활성화하지 않는다. 로컬 관심 저장과 Web Share·ICS는 계속 동작한다.
